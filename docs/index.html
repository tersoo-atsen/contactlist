Installation

Clone the project: git clone git@github.com:tersoo-atsen/contactlist.git
Change directory: cd contactlist
Install node modules: npm install
Start the server: npm start
Run tests: npm test
To see the app locally, visit: npm run docs To see the app locally, visit: http://localhost:5000/
